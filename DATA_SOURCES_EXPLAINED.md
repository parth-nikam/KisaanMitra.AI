# Data Sources Explained - KisaanMitra

## Current Data Sources by Feature

### 1. Market Prices (MARKET Agent)
**Source**: Static data in `market_data_sources.py`
**NOT using**: AgMarkNet API (too slow - 2-5 seconds)

**How it works:**
- Hardcoded price data for 8 major crops (wheat, rice, cotton, onion, potato, tomato, sugarcane, soybean)
- Updated manually/weekly
- Instant response (<100ms)
- Includes: average/min/max prices, trends, top mandis

**Data Location**: `src/lambda/market_data_sources.py` → `STATIC_MARKET_PRICES` dictionary

**Example:**
```python
"wheat": {
    "average_price": 2450,
    "min_price": 2200,
    "max_price": 2600,
    "trend": "stable",
    "top_mandis": [
        {"name": "Mumbai APMC", "price": 2500},
        {"name": "Pune APMC", "price": 2450}
    ],
    "last_updated": "2026-02-26"
}
```

**Confirmation**: Look for log: `[INFO] ✅ Using static market data for {crop}`

---

### 2. Crop Budgets (FINANCE Agent)
**Source**: AI-generated by Claude Sonnet 4
**NOT using**: Any external API

**How it works:**
- Claude Sonnet 4 generates budgets based on its training data
- Uses knowledge of Indian agricultural economics
- Considers regional variations
- Generates realistic estimates based on 2026 market conditions

**Is it accurate?**
- ⚠️ **AI-estimated values** - Not from real-time database
- ✅ Based on Claude's training data (up to April 2024)
- ✅ Considers regional factors
- ⚠️ May not reflect exact current prices
- ✅ Good for planning and estimation

**Confirmation**: Look for log: `[DEBUG] Model: us.anthropic.claude-3-5-sonnet-20241022-v2:0`

---

### 3. Crop Disease Detection (CROP Agent)
**Source**: Kindwise Plant.id API (external, real-time)

**How it works:**
- User sends crop image
- Image sent to Kindwise API
- Real AI-powered disease detection
- Returns disease name, confidence, treatment

**Confirmation**: Look for log: `Analyzing image with Kindwise API...`

---

### 4. Government Schemes (FINANCE Agent)
**Source**: Hardcoded in Lambda function

**How it works:**
- Static list of schemes (PM-KISAN, PMFBY, KCC, etc.)
- Manually maintained
- Not fetched from government API

**Data Location**: `match_government_schemes()` function

## The Budget Data Concern

### Current Reality:
**Budget values are AI-generated estimates, NOT from AgMarkNet or any real-time API.**

### Why?
1. AgMarkNet API is extremely slow (2-5 seconds per request)
2. No reliable real-time budget/cost API exists for Indian agriculture
3. AI provides reasonable estimates based on training data

### Accuracy Level:
- **Market Prices**: ⚠️ Static data (updated weekly manually)
- **Crop Budgets**: ⚠️ AI estimates (not real-time)
- **Feasibility**: ✅ AI analysis (good for general guidance)
- **Disease Detection**: ✅ Real-time API (Kindwise)
