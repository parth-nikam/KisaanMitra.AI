## Troubleshooting

### If "Site Can't Be Reached" Error:
1. **Clear browser cache** - Hold refresh button and select "Hard Reload"
2. **Try incognito/private mode** - Bypasses cache
3. **Check internet connection** - Make sure you're connected
4. **Try different browser** - Chrome, Safari, Firefox
5. **Wait 1-2 minutes** - S3 cache may need to clear
