## How It Works Now

### Weekly Training Flow
```
1. EventBridge triggers Lambda (Sunday 2 AM IST)
   ↓
2. Lambda fetches 5 years of data from S3
   ↓
3. Lambda starts SageMaker AutoML job
   ↓
4. SageMaker tests 6 algorithms:
   - ARIMA
   - ETS (Exponential Smoothing)
   - Prophet
   - NPTS (Non-Parametric Time Series)
   - DeepAR+
   - CNN-QR (Convolutional Neural Network)
   ↓
5. SageMaker picks best model (based on metrics)
   ↓
6. Model saved to S3
   ↓
7. [MANUAL] Generate forecasts using model
   ↓
8. Store forecasts in DynamoDB
   ↓
9. WhatsApp bot queries DynamoDB for forecasts
```

### Farmer Query Flow
```
1. Farmer sends: "टमाटर का भाव कल क्या होगा?"
   ↓
2. WhatsApp bot receives message
   ↓
3. AI routes to Market agent
   ↓
4. Market agent detects price forecast query
   ↓
5. Calls handle_price_forecast_query()
   ↓
6. Queries DynamoDB: kisaanmitra-price-forecasts
   ↓
7. Fetches 30-day forecast for tomato
   ↓
8. Formats response in Hindi
   ↓
9. Sends to farmer via WhatsApp
```
