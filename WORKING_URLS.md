## Why HTTP URL Doesn't Work on Mobile

The S3 website endpoint uses HTTP (not HTTPS), and many mobile networks/browsers block HTTP connections for security reasons. This causes the "ERR_CONNECTION_RESET" error.

**Solution:** Use the HTTPS S3 direct URL instead.
